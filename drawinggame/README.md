## How to Use

1. Open `index.html` in your web browser
2. Select a color using the color picker or click a color swatch
3. Adjust the brush size using the slider
4. Click and drag on the canvas to draw
5. Use the Erase button to switch to eraser mode
6. Click Save to download your drawing
7. Click Clear to start over

## Controls

- **Draw Mode**: Click the Draw button (default) and drag to draw
- **Erase Mode**: Click the Erase button and drag to erase
- **Color Picker**: Click the color box to choose any color
- **Color Swatches**: Click any swatch for quick color selection
- **Brush Size**: Use the slider to adjust brush thickness
